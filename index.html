About Me

Hello! My name is Wesley, and I enjoy all things web related. As someone who transitioned from working in logistics to a more technical role, I've come to appreciate the details on websites that provide a smoother experience in the UI for the user.


My interest for web development began when I wanted to create a website for my photography business, creating a website using sites like Wix to building one from scratch. There was that sense of accomplishment when the website looks just like how you wanted it to.
